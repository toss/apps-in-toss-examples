`Storage`를 사용해 입력한 텍스트의 초안을 자동으로 저장하고, 앱을 다시 실행해도 이전에 작성하던 내용을 불러올 수 있는 예제에요.  
모바일 앱의 로컬 저장소를 활용하면 사용자의 데이터를 영구적으로 유지할 수 있어요.

## 🚀 설치 및 실행 방법

1. **ZIP 파일**을 다운로드하고 압축을 풀어주세요.

2. 필요한 패키지를 설치해요.

   ```
   yarn install
   ```

3. 개발 서버를 실행해요.

   ```
   yarn dev
   ```
